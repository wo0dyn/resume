Installation
============

Requirements
------------

* pdfTeX 3.14159265-2.6-1.40.21 (TeX Live 2020) or newer
* Python 3.8+

Create PDF (pdflatex)
---------------------

.. code-block:: console

    $ pip install invoke      # Install simple task manager in python
    $ inv                     # Default task will build the LaTeX source
